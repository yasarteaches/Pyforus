🔧 Lesson 4: Functions

🌟 Learning Objectives

  • Understand what functions are and why we use them
  • Learn how to define and call a function in Python
  • Explore parameters and return values

💻 Code Examples

# Defining a function
def greet():
    print("Hello, welcome!")

greet()

def greet_user(name):
    print("Hello, " + name + "!")

greet_user("Mira")

def add(a, b):
    return a + b

result = add(3, 4)
print(result)

📝 Your Assignment

def square(n):
    return n * n

print(square(7))
