🔤 Lesson 3: Strings & Numbers

In Python, strings are words or text, and numbers are — well — numbers! 😄

You can use them together in print statements or math operations.


# String example
name = "Yasar"
print("Hello, " + name)

# Number example
x = 5
y = 3
print("The total is:", x + y)

Try editing the name or the numbers and run this in Replit or your code editor!
