The turtle module lets you control a turtle that draws on the screen. You can move it forward, turn it, change colors, and much more!

ðŸ’» Example Code

import turtle

window = turtle.Screen()
my_turtle = turtle.Turtle()

my_turtle.forward(100)
my_turtle.left(90)
my_turtle.forward(100)

window.mainloop()

This code creates a window and makes the turtle draw a simple L-shape.
